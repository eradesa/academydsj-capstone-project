Frequently Asked Questions

StudyMate is a full-stack web app that helps you manage study notes, get AI-powered summaries, and connect with Claude Desktop through an MCP server.

Click the "Summarize" button on any note. The app sends your note to OpenAI, which returns 3 bullet-point summaries and a quiz question.

MCP (Model Context Protocol) lets AI assistants like Claude Desktop access external tools. Our MCP server lets Claude list and create your notes.
